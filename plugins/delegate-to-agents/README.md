# delegate-to-agents

Delegate coding and research tasks from Claude Code to external agent CLIs
(Codex, Gemini, Qwen, OpenClaude, OpenCode) and Claude subagents, with parallel
and patch-review patterns.

Full documentation, install instructions, and credits are in the
[repository README](https://github.com/josephyaduvanshi/delegate-to-agents).

Quick start once installed:

```
/delegate codex fix the failing auth test
/delegate --review gemini migrate the tests to pytest
```
